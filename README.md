# Astro Basic App

This is a minimal Astro app created as an example.

Quick start:

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev

# Build for production
npm run build

# Preview build
npm run preview
```

Notes:
- Requires Node.js and npm installed (Node 18+, npm 8+ recommended).
- This project uses `astro` in devDependencies; no adapter is used (static output).
